# Samples

This folder contains a complete static example for quick review.

- `three_chapter_novel.txt`: demo input with three chapters.
- `sample_adaptation_plan.md`: AI co-writer planning output generated before script generation.
- `sample_adaptation_plan.json`: structured adaptation plan used by the generator.
- `sample_script.json`: structured script JSON generated by the workflow.
- `sample_script.yaml`: editable screenplay YAML exported from the validated JSON.
- `sample_adaptation_report.md`: validation report generated by the workflow.

These files are generated from mock mode so reviewers can inspect the expected output shape without running an external model.
